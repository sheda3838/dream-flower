# Dream Flower

A beautiful web application built with React and Vite.

## Links

- **Demo:** [https://dream-flower.vercel.app/](https://dream-flower.vercel.app/)
- **Repository:** [https://github.com/sheda3838/dream-flower.git](https://github.com/sheda3838/dream-flower.git)

## Getting Started

To run this project locally:

1. Clone the repository: `git clone https://github.com/sheda3838/dream-flower.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
