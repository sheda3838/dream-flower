# Dream Flower

Dream Flower is a beautifully designed, responsive web application built to showcase premium floral services, event management, and custom booking experiences. It provides a seamless user journey with elegant animations and modern aesthetics.

## Features

- Fully responsive design optimized for desktop, tablet, and mobile devices.
- Interactive hero section with dynamic video background.
- Elegant service and gallery sections with smooth animations.
- Custom booking form for tailored user requests.
- Premium typography and highly curated color palette.
- Accessible and reusable component architecture.

## Technologies Used

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Installation Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/sheda3838/dream-flower.git
   ```
2. Navigate into the project directory:
   ```bash
   cd dream-flower
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Running the Project Locally

To start the development server, run:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

## Project Structure

- `src/components/`: Contains all reusable React components (e.g., Navbar, Hero, Services, Gallery).
- `src/assets/`: Stores images, videos, and static assets.
- `src/App.jsx`: Main application component managing the layout and sections.
- `src/index.css`: Global styles and Tailwind configuration base.
- `public/`: Static files such as favicons.

## Responsive Design Information

The application is built with a mobile-first approach using Tailwind CSS. It fluidly adapts to various screen sizes, ensuring consistent spacing, typography readability, and interactive element usability across all platforms.
